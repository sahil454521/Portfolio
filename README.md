# Portfolio Website

My personal portfolio website showcasing my projects and skills.

## Tech Stack
- React 
- Vite
- Tailwind CSS
- React Router DOM
- React Icons

## Features
- Responsive Design
- Dark Theme
- Project Showcase
- Skills Animation
- Contact Information

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/sahil454521/portfolio.git
```

2. Install dependencies:
```bash
cd portfolio
npm install
```

3. Run development server:
```bash
npm run dev
```
